How to Deploy an App to Github

  1. Create a new repository on Github
  2. Using the command line, initiate git: git init
  3. Optional: Create a README file: git add README.md
  4. Add the files of the repository: git add .
  5. Commit the files of the repository: git commit -m "{comment}"
  6. Set up and push the repository via the command line using the commands provided from Github when the repository was initially created:
    • git branch -M main
    • git remote add origin https://github.com/{gitname}/deploying-react-app.git
    • git push -u origin main
